# Backend:
Node.JS and MongoDB (using moongoose currently)

```bash
npm i `
```

run backend in dev:

```bash
npm start 
```

# Frontend:

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
